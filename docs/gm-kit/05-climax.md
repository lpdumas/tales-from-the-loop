# Section 5 — Le Climax : L'Infiltration du Loop

---

## L'Enjeu : Compte à rebours

La brèche principale **pulse**. Chaque pulsation l'élargit.

| Pulsation | Effet dans le monde réel | Tours restants |
|-----------|--------------------------|----------------|
| 1re | Glitchs permanents dans un rayon de 1 km autour du Loop | **12 tours** |
| 2e | Le ciel devient violet, les gens ordinaires commencent à voir les anomalies | **8 tours** |
| 3e | Les doubles apparaissent en masse, la Nuée envahit les rues | **4 tours** |
| Finale | **Fusion** — le Vide et notre monde se superposent. Game over. | **0** |

> **Minuteur** : Chaque tour = ~2 minutes en jeu. Le compteur avance d'1 pulsation chaque fois que les PJs échouent à un jet critique ou perdent trop de temps. Ajuster selon le rythme de la table.

### Condition d'échec
- Le compteur atteint 0 **OU** tous les PJs ont la Condition: Brisé
- Échec ≠ mort. Conséquence : le monde change irréversiblement, le Vide s'installe — les enfants survivent mais le monde ne sera plus jamais normal (setup pour la suite de la campagne)

## L'Infiltration

### Le Loop — Un Tombeau

L'Entité n'a plus besoin de ses sbires dans notre monde — elle les a **consommés** (drainé leur énergie vitale). Le Loop est pratiquement désert. Quelques employés corrompus restent pour garder les apparences, mais l'intérieur est un **charnier** : couloirs vides, lumières qui clignotent, corps effondrés aux postes de travail.

> **Ambiance** : Pas un bâtiment sécurisé plein de gardes. Un endroit mort. Le danger vient de la Nuée, des anomalies, et de l'horreur de la découverte — pas d'une armée.

### Accès au Loop
- **Entrée secrète** (révélée par le journal codé) : Tunnel de maintenance nord, derrière le poste de transformation électrique. Porte blindée avec verrouillage numérique — Erik n'avait pas le code.
  - **Hacker le code** : mini-jeu (Mastermind, séquence logique, ou wire puzzle — voir notes ci-dessous). Un jet de **Bidouille** (diff. 2) donne un indice (un chiffre révélé).
  - **Alternative** : le Briquet à Résonance fonctionne comme clé (diff. 1)
- **Badge d'Erik** : utilisable sur l'entrée principale (accès niveau 2) mais risque de croiser les quelques pantins restants

### Mini-jeu : Hacker la porte blindée
- **Mastermind** : code à 4 chiffres, les joueurs proposent des combinaisons, le MJ dit combien sont bons/bien placés. Chaque essai = 1 tour en jeu.
- **Wire puzzle** : dessiner 5-6 fils emmêlés sur une feuille, les joueurs tracent les connexions pour trouver l'ordre.
- **Séquence logique** : suite de chiffres/symboles avec un pattern à compléter.

### Obstacles dans les sous-niveaux

| Obstacle | Description | Solution |
|----------|-------------|----------|
| **Les corps** | Employés du Loop effondrés à leurs postes, drainés par l'Entité. Pas de blessures visibles — juste des visages figés. | Test d'**Endurer** (diff. 1) ou Condition: Effrayé. Le Spectro-Viseur montre des traces d'énergie aspirée. |
| **Pantins restants** (1-2) | Rares employés encore contrôlés, sourire figé, regard vitreux. Gardent les apparences. | **Se Faufiler** (diff. 2) pour les éviter, **Empathie** (diff. 2) pour détecter le contrôle. Le Spectro-Viseur les révèle. |
| **La Nuée (essaim)** | 2D6 feux follets errent dans les couloirs proches de la brèche | Disrupteur de Phase pour les nettoyer, **ou** surcharger le disjoncteur du couloir (**Bidouille** diff. 2) |
| **Anomalies de couloir** | Plus on s'approche de la brèche, plus la réalité se distord — gravité décalée, échos temporels, ombres qui bougent seules | Pas de solution — ambiance pure. Le Spectro-Viseur montre la densité de Vide qui augmente. |

## Le Choix Final

> La salle de la brèche. Un trou dans la réalité, violet et pulsant, flotte au centre d'un anneau de métal géant. Des câbles y convergent de partout. **Erik n'est pas là.** Il est piégé dans le **Nexus** — de l'autre côté de la brèche — avec son double.

### Le Nexus

Erik et son double sont coincés dans le Nexus, l'espace entre notre monde et le Vide. Le Module Nexus du bunker était le seul lien de communication (one-way, du Nexus vers nous). Le double d'Erik l'utilisait pour communiquer avec Erik avant que celui-ci ne soit piégé à son tour.

### Le Dilemme

**Fermer la brèche** = sauver le monde, mais **couper définitivement le lien avec le Nexus**. Erik reste piégé de l'autre côté. À moins de trouver un autre moyen...

| Option | Conséquence | Ce qu'il faut faire |
|--------|-------------|---------------------|
| **A — Fermer la brèche immédiatement** | La brèche se referme. Le monde est sauvé. Erik reste piégé dans le Nexus — peut-être pour toujours. | Surcharger le Disrupteur (**Bidouille** diff. 3) et tirer dans la brèche |
| **B — Tenter de ramener Erik d'abord** | Perdre **4 tours** pour modifier le Module Nexus en 2-way et établir un contact. Risque : le compteur avance dangereusement. | **Bidouille** (diff. 4) pour l'upgrade 2-way + **Calculer** (diff. 2) pour synchroniser le signal + **Bidouille** (diff. 3) pour la brèche |
| **C — Erik communique depuis le Nexus** | Si les PJs ont upgradé le Module en 2-way (ou s'ils trouvent un moyen de capter un signal), Erik les guide à distance. Moment émotionnel. | Erik donne **+2 à tous les jets** de la séquence finale. La brèche se ferme. Erik reste dans le Nexus mais le lien 2-way permet un espoir futur. |

> **Note au MJ** : L'Option C est le "filet de sécurité" narratif. Ne la proposer que si les joueurs sont vraiment bloqués ou si le moment émotionnel le justifie. Laisser les joueurs choisir A ou B d'abord. L'Option B est quasi-impossible sous pression du compteur — c'est voulu. L'espoir de ramener Erik peut devenir un arc narratif pour la suite de la campagne.

## Boss Fight : L'Entité

L'Entité n'est PAS un monstre qu'on peut frapper. C'est une **force**. Le "combat" est une course contre la montre avec des obstacles.

### Principe : Kids on Bikes, pas D&D
- **Pas de points de vie**. Pas de combat mortel.
- Les PJs accumulent des **Conditions** (Effrayé, Épuisé, Blessé) à chaque complication
- Un PJ avec toutes les Conditions = **Brisé** — il ne peut plus agir mais n'est pas mort

### Séquence du boss fight (3 phases)

#### Phase 1 — La Nuée Finale
- **2D6 feux follets** attaquent en vagues
- Le Disrupteur les élimine mais **attention à la surchauffe**
- Les PJs sans gadget doivent trouver des solutions créatives (disjoncteurs, aimants d'urgence dans la salle)

#### Phase 2 — Les Visions du Vide
- L'Entité attaque **psychologiquement** — chaque PJ voit sa pire peur
- Test d'**Endurer** (diff. 2) pour chaque PJ. Échec = Condition: Effrayé
- Le Spectro-Viseur aide : +2 au test (les visions sont identifiées comme fausses)

#### Phase 3 — Fermer la brèche
- Le porteur du Disrupteur doit le **surcharger** (test de **Bidouille** diff. 3)
- Un autre PJ doit **maintenir l'anneau stable** (test de **Calculer** diff. 2)
- Un troisième PJ doit **couvrir** les deux autres contre la Nuée restante
- Si un PJ est Brisé, les autres peuvent prendre le relai mais avec **+1 de difficulté**

### Résolution

| Résultat | Narration |
|----------|-----------|
| **Réussite totale** (tous les jets passent) | La brèche se referme dans un flash de lumière blanche. Le Loop s'éteint. Silence. Le monde redevient normal. |
| **Réussite partielle** (1-2 échecs) | La brèche se ferme mais laisse des **cicatrices** — certaines zones gardent des propriétés étranges. Le Loop est endommagé. Des questions restent. |
| **Échec** | La fusion a commencé. Les PJs s'échappent. Le monde a changé — le Vide est désormais partiellement superposé. Nouveau statu quo pour la suite de la campagne. |
