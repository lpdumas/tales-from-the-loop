# GM Kit — Tales from the Loop : Projet Janus

> **Thème** : Horreur cosmique légère · Science dévoyée · Héritage
>
> **Synopsis** : Erik (père de Thor) a découvert que le Loop, via le "Projet Janus", ouvre des brèches vers un "Vide" habité par une Entité dévoreuse. Capturé par le Loop, il a laissé un bunker, un message incomplet et des gadgets bricolés. Les enfants doivent réparer le message, trouver 3 composants rares, et fermer la brèche principale avant la fusion des mondes.

## La Menace en bref : "La Nuée"

- **Nature** : Feux follets magnétiques — résidus d'énergie du Vide
- **Comportement** : Attaquent les **objets** des PJs (tech ou pas — impulsion EM)
- **Dégâts** : Brisent/grillent un objet au hasard sur un PJ
- **Piège** : Tirer au Disrupteur les dissipe, mais **ils réapparaissent** tant que la brèche source est ouverte
- **Vraie solution** : Localiser la brèche (Spectro-Viseur) et la colmater (Disrupteur ciblé sur la brèche)
- **Peur** : 1re rencontre = jet d'**Endurer** ; objet perso brisé = jet d'**Endurer**

## Règle clé : "Ne faites confiance à personne"

- L'Entité peut **contrôler la conscience** de n'importe qui (adultes surtout)
- Détection : **Empathie** (diff. 3) ou **Spectro-Viseur** (automatique, aura violette)
- Les PJs apprennent cette info au **1er composant installé** dans le terminal
